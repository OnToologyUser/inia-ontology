Supersetc Volver a índice o Índice de clases

IRI: http://dataid.dbpedia.org/ns/core#Superset

This dedicated version of a dataid:Dataset has exacly one purpose: to point out all its Sub-Datsets with void:subset. A dataid:Superset has no data itself and is therefore prohibited to point out Distributions with dcat:distribution. It can be used in a dataset hierarchy (e.g. as a root dataset), or as a container for other datasets.
Definida por
http://dataid.dbpedia.org/ns/core#
tiene superclases
Dataset c
distribution op max 0

aperturaop Volver a índice o Índice de propiedades de objetos

IRI: http://dataid.dbpedia.org/ns/core#openness

Descripción general de cómo se compartirán los datos. Por ejemplo, períodos de embargo (si los hay), esquemas de mecanismos técnicos para diseminación o una definición de si el acceso será ampliamente abierto o restringido a grupos específicos. En caso de que el conjunto de datos no se pueda compartir, se deben mencionar los motivos (por ejemplo, normas éticas, de datos personales, propiedad intelectual, comercial, relacionadas con la seguridad, relacionadas con la seguridad).
