Introducción volver a índice

La ontología para representar el los metadatos de los conjuntos de datos publicados por la organización INIA es un extensión de las ontología dataid. Aunque la mayor parte del modelado se ha reutilizado de dataid, algunos conceptos y relaciones específicas para el caso de los conjuntos de datos de INIA han sido definidas, por ejemplo para indicar el número de elementos o el software apropiado para tratar los conjuntos de datos.
